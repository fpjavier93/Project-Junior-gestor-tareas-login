## Encoding

Los archivos del proyecto deben tratarse como UTF-8.

Cuando Codex lea archivos con acentos desde PowerShell, especialmente archivos `.md`, debe usar:

```powershell
Get-Content ruta\archivo.md -Encoding UTF8
```
